How to use Controller

Credit : angular org

Controller ใน Angular JS เราสามารถสร้าง ให้เกิดเป็น nested controller ได้โดยมี parent และ child ได้และ controller นั้นเราจะสามารถเขียนได้สองแบบ
1. Basic Controller
 การทำงานของ controller จะมีการทำงาน ในฝั่งของ Code Javascript ซึ่งเราจะใช้ในการควบคุมขอบเขตของ scope เพื่อน เเบ่งการทำงานของ view-model หรือ binding-template ให้เป็นสัดส่วน หรือใช้งานร่วมกัน
  • Script.js
  • HTML

angular.controller('TestController', function ($scope) {

$scope.data = { };

$scope.data.content = ' World !!!'

}]);

<div ng-controller="TestController">

Hello {{data.content}}

</div>

Hello {{data.content}}

จากตัวอย่างข้างต้นเราจะสงเกตุได้ว่า มีการประกาศ controller "TestController" ที่ฝั่ง angular.js และในฝั่งของ view นั้นมีการตั้ง attribute ให้กับ div โดย set ng-controller="TestController"

2. Controller Custom
 การทำงาน Custom Controller จะเป้นการทำงานตาม Directives โดยเราสามารถฝัง controller เข้าไปใน Template ต่างๆได้เหมาะแก่การใช้นการทำ StateView หรือ Directives
