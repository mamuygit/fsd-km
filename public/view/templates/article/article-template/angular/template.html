Template

Credit : angular org

ใน angular คำว่า Template นั้นก็คือ file html นั้นเอง เพียงแต่ html file เหล่านั้น จะมี markup ที่จะใช้ใน angular เพิ่มเข้ามา

Static Template
  โดยปกติเวลาเราจะสร้าง HTML เราจะเขียน เป็น static HTML ขึ้นมา 1 หน้าเพื่อให้ง่ายกับการเริ่มต้นใช้งาน angular ให้สร้าง index.html ขึ้นมา 1 file โดยใส่
code :
<div> Hello world !!! </div>

result :
Hello World !!!

Angular Templates
  ทีนี้เรามาดูการวาง html template สำหรับใช้งานกับ angular กันมั่งการเขียน angular นั้นจะเขียนในรูปแบบ pattern ของ MVC โดยในส่วนของ html เราจะถือเป็น view โดย angualr จะมอง view เป็นเสมือน presentation ส่วนแสดงผล ดังนั้นเมื่อใดที่มีการเปลี่ยนเเปลงข้อมูลในส่วนของ model angylar จะมีการโหลดข้อมูลที่มีส่วนที่เกี่ยวข้องกับส่วนของ binding และทำการ update view จากไฟล์เดิมที่สร้างใน section ก่อนหน้า [Static Template] ให้เปลี่ยนแปลง code เป็นดังนี้
code :

<html>

<head>

<script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.5.3/angular.min.js"></script>

</head>

<body>

<div ng-model="world" ng-init="world='World'"> Hello {{world}} !!! </div>

</body>

</html>


result :
Hello {{world}} !!!
จากตัวอย่างข้างต้นนั้นดูเหมือน html ธรรมดากับ markup แบบใหม่โดยตัวอย่างนี้แสดงให้เห้นถึงการทำงานของ angular template แบบ basicly โดยเป็นการสร้าง model ที่มีชื่อว่า world และ set ค่าให้เป็น string 'world' ที่ ng-init จากนั้น ให้สังเกตุใน div ว่าจะมีสัญลักษณ์ {{...}} อันนี้คือ block ของ angular สำหรับการเรียกใช้ object ที่เรามีภายใน scope นั้นๆ ในที่นี้ภายใน scope มี object model ชื่อ world เราจึงสามารถเรียก {{world}} ได้

credit : https://docs.angularjs.org/tutorial/step_04

  สำหรับ Angular รูปแบบการทำ data binding เมื่อ model มีการ update data ฝั่ง view จะมีการ update view แบบอัตโนมัติโดยการทำงานนี้ไมไ่ด้มีส่วนเกี่ยวข้องกับ DOM.
