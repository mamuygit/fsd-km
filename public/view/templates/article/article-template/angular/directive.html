การทำ Directives คือการทำ dynamic template binding คือการการวางโครงสร้างของ view โดยสามารถเปลี่ยนแปลงตาม content ในแต่ละส่วนของเว็บที่มีการเกิดการเปลี่ยนแปลงได้อีกนัยนึงคือการทำ global tempalte เพื่อให้สามารถได้ใช้ได้หลายๆ ส่วนใน website การทำ Directive มีสองแบบ
1. Static html Directives

  • Script.js
  • HTML

angular.directive("myDirective", function () {

return {

controller: function ($scope) {

$scope.name = 'MyName',

$scope.address = 'My Address'

},

template: 'Name: {{name}} | Address: {{address}}'

};

});

 จากตัวอย่างข้างต้นจะเป็นการทำงานของ Directive ในรูปแบบของ static html ซึ่งการ Matching ระหว่าง Directive กับ View นั้นจะมีการระบุผ่าน ชื่อของ Directive นั้นๆกับ tag html ใน View จากตัวอย่างได้มีการสร้าง Directive myDirective ในฝั่งของ Code Javascript ซึ่งการตั้งชื่อจะมีการใช้รูปแบบของ caramel คือ word หน้านั้นจะใข้ตัวสะกดตัวเล็กและ word ที่ตามมาจะมี ตัวแรกเป็นตัวใหญ่จะ match กับ html tag ฝั่ง view ใน tag my-Directive
