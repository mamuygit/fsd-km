Scope

Credit : angular org

ใน Angular JS เราจะมีการใช้งาน Scope ได้สองรูปแบบ
 1. rootscope คือ ขอบเขต ทั้งหมดที่คลุมตั้งแต่ ng-app
 2. scope คือ ขอบเขต ทั้งหมดที่คลุมตั้งแต่ ng-controller
โดยการแบ่งสัดส่วนของ scope จะขึ้นอยู่กับ html ที่ จะนำมาใช้คู่กับการทำ data-binding

  • Script.js
  • HTML

angular.module('scopeExample', [])

.controller('MyController', ['$scope', function($scope) {

$scope.username = 'World';

$scope.sayHello = function() {

$scope.greeting = 'Hello ' + $scope.username + '!';

};

}]);

<div ng-app="scopeExample" ng-controller="MyController">

Your name:

<input type="text" ng-model="username">

<button ng-click='sayHello()'>greet</button>

<hr>

{{greeting}}

</div>

Your name:
{{greeting}}
จากตัวอย่างข้างต้น เป็นการเขียน angular โดยมี controller เป็นตัวคุม scope data
 - Script.js
  ได้มีการสร้าง Controller function ขึ้นมาโดยมีชื่อว่า 'MyController' โดยภายใน Controller นี้ประกอบไปด้วย variable username มี value เป็น 'World' และมี function 'sayHello'
 - html
 ภายใน บริเวณของ Controller 'MyController' ใน view หรือ html file เราจะสามารถเรียกใช้งาน variable function ทุกอย่างที่เรไาด้ทำการประกาศเอาไว้แล้วได้

**** ปล. view ภายใน attribute ng-*[Whatever] เราไม่จำเป็นจะต้อง ใช้สัญลักษณ์ {{...}} หรือ block ดังนั้นจึงใช้ ng-model="username" ได้ สรุปๆง่ายๆคือ ถ้าฝั่ง view มีการจะเรียกใช้ variable หรือ fucntion ภายใน Scope ของ Controller นั้นๆ ถ้าจะใช้คู่กับ HTML tag โดยใช้ attribute ng-*[Whatever] ไม่จำเป็นต้อง ใช้ block แต่ถ้าใช้ในการ binding-html จะต้องมี block
