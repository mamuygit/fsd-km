Angular JS คือ Javacript Framework แบบหนึ่งหมือนกับ Jquery ที่พัฒนาขึ้นมาช่วยในการพัฒนาเว็บไซต์ให้นักพัฒนาสามารถ พัฒนาเว็บไซต์ได้รวดเร็วและมีคุณภาพมากขึ้น Angular JS มีรูปแบบการทำงานเป็น MVC ในรูปแบบของ MVW

What is a MVW framework?
  MVW (Model-View-Whatever) เป็นการแบ่งการทำงานระหว่างฝั่งของ business logic หรือ model และ ฝั่ง presentation เป็น view ซึ่งเราจะออกแบบ application ของเราให้มีการทำงานในเชิง view-model ก็ได้หรือใช้งาน controller ก็ได้

Conceptual Overview
  Angular มี concept ที่หลากหลาย ใน framework ในตัวมันเองในที่นี้จะยกมาคร่าวๆ แต่ถ้าอยาก เรียนรู้เพิ่มเติมสามารถเข้าไปได้ที่ angular.org
Template
HTML with additional markup
Controller
the business logic behind views
Scope
context where the model is stored so that controllers, directives and expressions can access it
Model
the data shown to the user in the view and with which the user interacts
Filter
formats the value of an expression for display to the user
View
what the user sees (the DOM)
Data Binding
sync data between the model and the view
Expressions
access variables and functions from the scope
Service
reusable business logic independent of views
Directives
extend HTML with custom attributes and elements
